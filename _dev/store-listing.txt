CHROME WEB STORE LISTING — Scholar Journal Filter v1.1.0
========================================================

DESCRIPTION (paste into Description field):

Filter Google Scholar by journal — on both search results and author profile pages.

Set up a whitelist (or blacklist) of target journals for your field. The extension highlights matching papers with a green border, dims non-matching ones, or hides them entirely. Ships with 82 pre-loaded journals across management, psychology, sociology, general science, and the FT 50 — fully customizable.

SEARCH RESULTS FILTERING
Run any Google Scholar search. Papers from your target journals are instantly highlighted. Non-matching results and non-journal items (books, theses, working papers) can be shown, dimmed, or hidden independently.

PROFILE PAGE FILTERING
Visit any author's Google Scholar profile to see which of their publications appear in your target journals. Useful for quickly scanning a colleague's or reviewer's output for work in specific outlets.

HIGHLY CITED BADGE
Optional gold badge on papers above a configurable citation threshold (default: 500). Highly cited papers stay visible even when their journal isn't in your list — so you never miss influential work.

LITERATURE REVIEW EXPORT
Select papers with checkboxes across multiple pages, then batch export to Excel. Each export includes title, authors, journal, year, citation count, and PDF availability. Optionally enrich with full abstracts and DOIs via the free OpenAlex API — no account required.

CUSTOMIZABLE JOURNAL LIST
Add, edit, or remove journals through the options page. Right-click any search result to add its journal instantly. Each journal supports multiple aliases to handle Google Scholar's inconsistent abbreviations.

PRIVACY
Runs entirely in your browser. No tracking, no analytics, no data collection. The only network request is the optional OpenAlex abstract lookup, which is anonymous.
